Ubuntu QML toolkit received support for Ubuntu 12.04 LTS and Ubuntu 13.04 [unread]
Wed, 16 Jan 2013 21:06:37 +0000
Tweet

Immediately after the arrival of the new 2013 year, Canonical revealed the polished and innovative Ubuntu phones OS, starting a serious exciting movement towards incarnating the pure Ubuntu desktop into a phone form-factor.

Ubuntu QML toolkit immediately followed after the successfully-received-by-users phone OS, toolkit that, while centering QML and utilizing the latest Qt technologies, allows (work in progress) third-party developers to create modern applications with fancy shapes, interesting animations, vivid colors, etc.

In the Ubuntu community, there are numerous user types, including users who cherish and daily use the LTS version of Ubuntu, users who have expressed demands for Ubuntu QML toolkit's availability for Ubuntu 12.04 LTS (since the initial QML toolkit announcement).

The developers have released a new Ubuntu QML toolkit version, introducing support for Ubuntu 12.04 and Ubuntu 13.04, thus, along with the initially-supported Ubuntu 12.10, reaching the majority of Ubuntu users.

The QML components (available via the above mentioned PPA) depend on the latest Qt versions, yet, the required Qt PPA has been updated also with support for Ubuntu 12.04 LTS, essentially, the LTS users are able to enjoy a full no-compromise experience (from a number of provided packages point-of-view).

Full installation details are available on http://developer.ubuntu.com/get-started/gomobile/

Everpad 2.4 released with new icons, table editing and email support [unread]
Wed, 16 Jan 2013 16:54:15 +0000
Tweet

Evernote is a popular online service that presents itself as an easy-to-use note-taking Web tool.

Everpad centers Evernote's power into a desktop-integrated filled-with-functionalities application, allowing users to hassle-free deal with, manage and edit notes, process performed via handy appindicator, enhanced Unity launcher quicklist and dedicated lens.

Formatting text, adding attachments, inserting tables, search-through text pieces are actions available via Everpad.

Everpad has been updated to another interesting release, 2.4, introducing new changes and functionalities.

Launching the newly released 2.4 version the user is to notice new icons, used for appindicator, app icon and inside its lens.

The icons used for the appindicator are monochrome with a square shape and symbolic attitude, adding a fresh look & feel to the Unity panel, as well as strengthening Everpad's note-taking identity.

For the app icon, Everpad comes now with a green square icon, properly occupying the Unity launcher.

Selecting a dark/white monochrome icon for its appindicator is as simple as navigating to Everpad appindicator-->Settings and Management-->Appearance and checking/unchecking Black tray icon.

Hitting the Create Note entry (available via its Unity launcher quicklist, too) summons the note-creation dialog, clicking on the toolbar's Insert Table, adds a table to the main view.

Everpad 2.4 introduces support for editing tables, meaning, after creating a table, right-clicking on a table-->Change table opens a dialog, thus allowing the user to modify the table's values (such as Rows, Columns, Width), clicking on the OK button, renders the table with the previously-edited values.

Along with numerous features, Everpad comes with attachment support, thus the user is able to add files (like for example, text files and images) as attachments to notes, attachments located and organized on the bottom area (of specific notes).

The new 2.4. Everpad enriches its attachment functionality, providing the ability to insert attachments into the actual note.

In order to insert an attachment into a piece of text, click on the text's place of insertion (per-user-preference), then right-click on the bottom-area's attachment-->Put to Content, action than inserts the attachment into a desired place (such as between two words, etc).

Furthermore, right-clicking on an inserted attachment-->Image Preferences allows users to modify the width and height of the rendered icon, tweaking the values to 20, decreases the icon size to 20.

A definitely interesting aspect of Everpad 2.4 is its send-by-email support, ability expressed through a newly implemented Email note button.

Sending a created note via email is to be achieved by clicking on the toolbar's Email note, action that opens the note via the default email client, furthermore, automatically adding the note's title (such as New note) to (Thunderbird's) Subject.

The mentioned features, along with fixes and optimizations, come to deliver a more refined and reliable Everpad release.

How do we install Everpad 2.4?
Add the following official PPA (Precise, Quantal)

sudo add-apt-repository ppa:nvbn-rm/ppa

sudo apt-get update
sudo apt-get install everpad
