Ubuntu One Music Store Plugin Dropped from Ubuntu 13.04
Fri, 18 Jan 2013 06:39:47 PST

Ubuntu’s default music player won’t ship with the Ubuntu One Music Store plugin in 13.04.

With the Ubuntu One Music Store now available via the web, and all Dash music recommendations linking there, the decision has been taken to retire the Rhythmbox plugin ‘in favour‘ of the web store.

But this change will make the store more discoverable.

No More Music Store in Ubuntu 13.04

Plugins, Schmugins

The Ubuntu One Music Store plugin debuted in Ubuntu 10.04, and since that time it’s not changed a great deal. It also limited itself to just two media players: Rhythmbox and Banshee.

But now that a link to the Ubuntu One Music Store is pinned on the launcher of Ubuntu 12.10 by default – and is likely to remain in place in 13.04 - every user of Ubuntu can see and access it without needing to open (or even use) Rhythmbox.

Ubuntu One Features Remain

Despite the store bidding adieu other features of the Ubuntu One Rhythmbox plugin will remain.

This includes the “auto-sync” feature that automatically adds purchased tracks to your library.

Ubuntu 13.04 Adds New Launcher Options
Fri, 18 Jan 2013 04:44:00 PST

New options have been added to the Appearance Settings pane in Ubuntu 13.04.

With a click you can disable the ‘Workspace Switcher’ item, used to navigate between the four workspaces Ubuntu provides by default, and add a ‘Show Desktop’ item for quickly hiding windows to access files or folders on the desktop.

New Launcher Settings in Ubuntu 13.04

Both options were previously available to Ubuntu users via the ‘Experimental’ section of the Compiz Config Settings Manager.

The ‘Workspace Switcher’ checkbox arrives ahead of a planned UI change in Ubuntu 13.04 that will disable the item by default.

As such a quick, user-friendly way to re-enable it was required – and early on in development.

‘RC Mini Racers’ December’s Top Selling Ubuntu App
Fri, 11 Jan 2013 08:34:53 PST

Backyard karting game ‘RC Mini Racers‘ was Ubuntu’s top selling paid application in December. 

The game managed to better its November position of #2.

RC Mini Racers Is A Blast To Play

Elsewhere in the chart slick desktop weather application Stormcloud storms (pun fully intended) into second place. It’s a surprising result for an application that was, until last month, available for free.

The paid version of Stormcloud, which we reviewed late last month, also featured in our Top 10 New Apps of 2012.

Other Placings

On the movers list we see last monthspopular games fall. Amnesia: The Dark Descent tumbles from #1 to #3; Bastion goes from #3 to #6; and Machinarium slides one from #7 to #6.

Limbo falls out of the Top 10 entirely.

But it’s not all downwards. Fluendo DVD Player, which offers a “legal” way to enable DVD playback in Ubuntu, re-enters the chart for the first time since September.

And another of last months reviews, Dare Up! Wingsuit, also makes the list at #10.
